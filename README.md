# wallet-bundle #

Package the WalletConnect functions into bundle that can be used in HTML file.<br>

Using
- [webpack](https://webpack.js.org/)
- [certhis wallet](https://github.com/certhis/CERTHIS-WALLET)

## Build ##
Install NPM modules.
```sh
npm i
```
Build bundle file.
```sh
make
```
Clean bundle file.
```sh
make clean
```

## Run ##
Open `test.html` in browser.
